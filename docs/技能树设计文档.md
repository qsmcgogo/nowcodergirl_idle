# 技能树设计文档

本文档记录当前技能树结构、解锁规则与显示规则。技能树使用灵感作为核心升级资源，定位是长期永久成长与结构化玩法解锁，不直接替代建筑、卡片或学科关卡奖励。

## 一、解锁与显示规则

- 首次获得灵感后，解锁技能树页签与第一层。
- 每一层解锁前，只显示该层锁定块。
- 鼠标悬浮锁定块显示解锁条件。
- 当前层解锁后，显示该层节点，并显示下一层锁定块。

## 二、第一层

| 节点 | 解锁条件 | 每级效果 | 消耗 |
|------|----------|----------|------|
| 语法 | 首次获得灵感 | 知识点 +0.1，手速 +0.1，耐力 +1 | 灵感 + 专注力 |
| 数学 | 首次获得灵感 | 思维 +0.2，知识点 +0.2 | 灵感 + 思维力 |

第一层升级成本倍率为 `1.218`。

## 三、第二层

第二层总解锁条件：语法 Lv.5 且数学 Lv.5。

| 节点 | 前置 | 每级效果 | 消耗 |
|------|------|----------|------|
| 顺序结构 | 无额外前置 | 手速 +0.08，耐力 +0.5 | 灵感 + 专注力 |
| 变量类型 | 语法 Lv.10 | 知识点 +0.15，码力 +0.05 | 灵感 + 专注力 + 思维力 |
| 加法 | 无额外前置 | 思维 +0.15，知识点 +0.05 | 灵感 + 思维力 |
| 减法 | 加法 Lv.5 | 手速 +0.12，耐力 +0.8 | 灵感 + 思维力 + 专注力 |

## 四、视觉结构

```text
语法                         数学
 | \                         |
 |  \                        |
 v   v                       v
顺序结构    变量类型        加法  ->  减法
```

实现中为了适配紧凑面板，会保持：

- 语法向左下连接顺序结构。
- 语法向右下连接变量类型。
- 数学向下连接加法。
- 加法向右连接减法。
