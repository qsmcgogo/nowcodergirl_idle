# 第一章建筑与资源节奏设计

本文档用于设计第一章的早期建筑、资源和解锁节奏。目标是解决“第一章推进过慢，并且中途只有自动练习一个明显解锁”的问题。

核心原则：

- 第一章不只靠削弱敌人数值，而是提高新系统出现密度。
- 玩家每 3-5 分钟应看到一个新目标、一个小建筑或一次明确成长。
- 建筑默认不设等级上限。建筑效果线性增加，指数成本和后续指数敌人成长负责控制长期性价比。
- 第一章目标时长：主动玩家约 20-35 分钟，轻挂机玩家约 45-75 分钟。

---

## 一、当前问题

当前新档早期结构大致如下：

| 节点 | 当前内容 | 问题 |
|------|----------|------|
| 思维力达到 10 | 解锁专注力 | 只有第二个基础资源 |
| 思维力、专注力都达到 10 | 展示研究区 | 仍需继续攒资源 |
| 消耗 20 思维力 + 20 专注力 | 解锁练习 | 早期第一次大消耗 |
| 首次战败 | 解锁跑步研究 | 战败前没有失败补偿 |
| 击败试卷 1 E 题 | 解锁自动练习研究 | 第一章中段唯一明显功能解锁 |
| 清完试卷 1 共 20 题 | 解锁第 2 图和数学训练 | 目标跨度太长 |

第一章慢的体感主要来自两个方面：

- 结构慢：清完 20 题前缺少分段奖励和新建筑。
- 恢复慢：初始血量低，战败后只能等待，失败没有转化为成长。

---

## 二、新资源设计

第一章建议只新增轻量资源，避免前期 UI 过度复杂。

| 资源 | 解锁位置 | 主要来源 | 主要用途 | 设计边界 |
|------|----------|----------|----------|----------|
| 草稿纸 | 击败 A 题 | 挂机少量产出、通关题目 | 升级草稿桌、安静角落、自动草稿机 | 低速通用材料，不直接进攻防乘区 |
| 错题笔记 | 首次战败 | 战败、首次挑战新题 | 升级错题本、小憩垫、失败补偿 | 鼓励失败成长，单次收益有限 |
| 复习券 | 击败 E 题 | 5 题节点奖励、战败概率、错题笔记兑换 | 解锁刷题模式、进入刷题模式、升级重复练习技能 | 只服务刷题循环，不进入普通建筑 |
| 记忆碎片 | 击败 E/J/O/T 题 | 每 5 题里程碑、第一章通关 | 解锁建筑、章节奖励、第二图预备建筑 | 第一章内半稀缺，用于控制系统开放 |

灵感保持纯粹：只用于升级技能树，不再作为建筑、刷题模式或其他研究的消耗。暂不建议第一章新增太多主资源。星星可以作为后续评价资源恢复，但第一版先不强依赖。

## 三、建筑列表

### 3.1 草稿桌

定位：提供思维力自动收集，让玩家在主动做其他事时仍有基础思维力入账。

| 字段 | 建议 |
|------|------|
| 解锁位置 | 思维力首次达到 10 |
| 初始建造成本 | 15 思维力 |
| 升级资源 | 思维力、草稿纸 |
| 基础成本 | 15 思维力 |
| 成本倍率 | 1.42 |
| 效果 | 思维力自动收集 |
| 效果公式 | `+0.08 * level /s` 自动获得思维力 |
| 第一章推荐等级 | 4-7 |

说明：草稿桌不提高主动收集思维力的效率。它提供一条独立的自动收益线，让“主动收集”和“建筑挂机”可以分开成长。

### 3.2 安静角落

定位：提供专注力自动收集，降低练习、跑步、自动练习和技能树之间的资源冲突。

| 字段 | 建议 |
|------|------|
| 解锁位置 | 专注力首次达到 10 |
| 初始建造成本 | 12 专注力 + 8 思维力 |
| 升级资源 | 专注力、思维力、草稿纸 |
| 基础成本 | 12 专注力 + 8 思维力 |
| 成本倍率 | 1.45 |
| 效果 | 专注力自动收集 |
| 效果公式 | `+0.07 * level /s` 自动获得专注力 |
| 第一章推荐等级 | 4-6 |

说明：安静角落不提高主动收集专注力的效率。专注力自动收益略慢于思维力，避免专注力过剩后跑步无脑常驻。

### 3.3 错题本

定位：把战败变成成长，不再只是等待回血。

| 字段 | 建议 |
|------|------|
| 解锁位置 | 首次战败 |
| 初始建造成本 | 免费解锁，赠送 3 错题笔记 |
| 升级资源 | 错题笔记、思维力 |
| 基础成本 | 5 错题笔记 + 10 思维力 |
| 成本倍率 | 1.5 |
| 效果 | 对当前未完成题目提供攻击加成 |
| 效果公式 | `+3% * level`，第一章推荐等级内约 `+9%` 到 `+15%` |
| 第一章推荐等级 | 3-5 |

说明：错题本是“卡关润滑剂”，不是永久攻击主成长。该加成只对当前未通关题生效，通关后清空。

### 3.4 小憩垫

定位：提升基础回血速度，缩短战败等待。

| 字段 | 建议 |
|------|------|
| 解锁位置 | 到达第 2 图 |
| 初始建造成本 | 6 错题笔记 + 15 专注力 |
| 升级资源 | 错题笔记、专注力 |
| 基础成本 | 6 错题笔记 + 15 专注力 |
| 成本倍率 | 1.48 |
| 效果 | 非跑步恢复速度增加 |
| 效果公式 | `+0.12 * level HP/s` |
| 第一章推荐等级 | 3-6 |

说明：基础回血从 `1/s` 逐步提高。跑步仍然有价值，因为跑步基础为 `3/s`，后续可让跑步吃小憩垫 50% 加成。

### 3.5 灵感灯

定位：稳定灵感掉落，降低技能树入口的坏运气，但不直接让灵感数量翻倍。

| 字段 | 建议 |
|------|------|
| 解锁位置 | 到达第 2 图，且已经获得过灵感 |
| 初始建造成本 | 10 错题笔记 + 20 思维力 |
| 升级资源 | 错题笔记、思维力 |
| 基础成本 | 10 错题笔记 + 20 思维力 |
| 成本倍率 | 1.55 |
| 效果 | 灵感掉落保底更稳定，小幅提升掉率 |
| 效果公式 | `灵感掉率 +1.5% * level` |
| 第一章推荐等级 | 1-3 |

说明：灵感灯只“稳”，不“爆”。第一章不建议提高单次灵感掉落量。

### 3.6 自动草稿机

定位：在正式自动练习前给一个半自动成长点，让 E 题前后不只有一个按钮。

| 字段 | 建议 |
|------|------|
| 解锁位置 | 到达第 3 图 |
| 初始建造成本 | 30 思维力 + 30 专注力 + 2 记忆碎片 |
| 升级资源 | 草稿纸、思维力、专注力、记忆碎片 |
| 基础成本 | 30 思维力 + 30 专注力 + 2 记忆碎片 |
| 成本倍率 | 1.6 |
| 效果 | 快进/离线时额外结算主线战斗效率 |
| 效果公式 | `+5% * level` 快进/离线主线战斗效率 |
| 第一章推荐等级 | 1-3 |

说明：自动草稿机不直接替代自动练习。它只对快进、离线或恢复期辅助生效。

### 3.7 记忆书架

定位：承接每 5 题里程碑，让第一章从“20 个题点”变成“四段进度”。

| 字段 | 建议 |
|------|------|
| 解锁位置 | 到达第 4 图 |
| 初始建造成本 | 3 记忆碎片 + 30 思维力 |
| 升级资源 | 记忆碎片、思维力、草稿纸 |
| 基础成本 | 3 记忆碎片 + 30 思维力 |
| 成本倍率 | 1.7 |
| 效果 | 全局攻击和血量小幅加成 |
| 效果公式 | `+1.5% * level` 全局攻击和血量 |
| 第一章推荐等级 | 1-4 |

说明：记忆书架提供很小的全局乘区，并设置软边界。它更像章节成就陈列，不应替代技能树。

### 3.8 备考时钟

定位：第一章后段为第二图数学训练铺垫，让玩家提前看到时间之砂和答题券循环。

| 字段 | 建议 |
|------|------|
| 解锁位置 | 到达第 5 图 |
| 初始建造成本 | 4 记忆碎片 + 50 专注力 |
| 升级资源 | 记忆碎片、专注力 |
| 基础成本 | 4 记忆碎片 + 50 专注力 |
| 成本倍率 | 1.65 |
| 效果 | 数学练习时间之砂固定加成，轻微降低答题券兑换成本 |
| 效果公式 | `时间之砂 +2 * level`，`答题券兑换成本降低 2% * level` |
| 第一章推荐等级 | 1-3 |

说明：第一章内主要展示预告，进入第 2 图后正式生效。

## 四、刷题模式与推关模式

第一章建议把主线战斗拆成两种模式：

| 模式 | 定位 | 是否推进题号 | 主要收益 |
|------|------|--------------|----------|
| 推关模式 | 当前默认模式，挑战当前未通关题目 | 是 | 灵感、草稿纸、记忆碎片、章节进度 |
| 刷题模式 | 重复刷已经通关的题目，用于卡关时补资源 | 否 | 草稿纸、少量错题笔记 |

### 4.1 刷题模式解锁

刷题模式本身作为一个需要研究的技能出现。

| 字段 | 建议 |
|------|------|
| 解锁位置 | 击败 E 题后出现研究 |
| 研究成本 | 20 思维力 + 20 专注力 + 2 复习券 |
| 入口位置 | 练习区战斗按钮旁，和推关/自动练习同级 |
| 默认目标 | 当前图已通关的最高题 |
| 可选目标 | 已通关题目，可向前选择低题 |

设计理由：

- E 题是第一章第一个明显中段节点，玩家此时已经理解战斗、战败、灵感和技能树。
- 如果玩家卡在 F-J 段，刷题模式能把“等待回血”变成“回头刷资源”。
- 刷题模式不替代推关，因为它不产出记忆碎片，也不解锁下一题。

### 4.2 刷题收益

刷题收益应比推关首通低，但稳定、可重复。

| 收益 | 建议公式 | 防爆限制 |
|------|----------|----------|
| 草稿纸 | `1 + floor(题号 / 5)` | 每次进入消耗复习券，避免无限空转 |
| 灵感 | 不掉落 | 灵感只来自推关/主线战斗，用于技能树 |
| 错题笔记 | 刷题胜利小概率 `+1`，刷题战败给失败补偿 | 只作为小额补偿，不作为主收益 |
| 复习券 | 不掉落 | 由推关节点、战败补偿或错题笔记兑换提供 |
| 记忆碎片 | 不掉落 | 只允许里程碑和章节通关给 |
| 答题券 | 不掉落 | 只允许章节/数学系统给 |

刷题模式最重要的是“补资源”和“补升级材料”，不是替代主线收益。

### 4.3 自动练习衔接

自动练习解锁后，建议允许玩家选择自动行为：

| 自动行为 | 说明 |
|----------|------|
| 自动推关 | 回满血后自动挑战当前未通关题 |
| 自动刷题 | 重复刷选定已通关题 |
| 智能模式 | 推关失败达到一定次数后自动刷题一段时间，再回到推关 |

第一版可以先实现手动切换“推关/刷题”，智能模式留到后续。模拟脚本中可以先用简单规则：当前题连续战败达到 2 次后，如果已学会刷题模式，则刷已通关最高题 60 秒，再回到推关。

### 4.4 刷题防爆原则

- 刷题不掉记忆碎片和答题券。
- 刷题不掉灵感，避免技能树被重复 farm 直接刷爆。
- 刷题不掉复习券，避免自己产门票形成永动。
- 同一题重复刷有递减收益。
- 刷题收益主要在加法区：草稿纸、少量错题笔记。
- 刷题技能可以继续升级，但升级只提高刷题效率，不直接提高推关战力。

| 阶段 | 预计时间 | 解锁内容 | 奖励/资源 | 设计目的 |

## 六、建筑线性成长与主动效率

### 6.1 建筑效果线性，成本指数

建筑第 `n` 级成本：

```text
cost(n) = ceil(baseCost * multiplier ^ (n - 1))
```

建筑效果统一采用线性成长：

```text
effect(level) = baseEffect * level
```

建筑第 `n` 级成本仍然指数增长。这样玩家可以直观看懂每一级提升，同时长期性价比会自然下降。

建议成本倍率：

| 建筑类型 | 成本倍率 |
|----------|----------|
| 基础资源产出 | 1.42-1.48 |
| 战斗补偿/回血 | 1.48-1.55 |
| 自动/跨系统收益 | 1.60-1.75 |
| 全局攻血乘区 | 1.70+ |

### 6.2 草稿桌和安静角落只给自动收益

草稿桌、安静角落不提高主动收集效率，只提供自动收集：

```text
思维力自动收益 = 0.08 * 草稿桌等级 /s
专注力自动收益 = 0.07 * 安静角落等级 /s
```

主动收集仍保持基础 `1/s`。这样玩家选择主动收集时仍有明确价值，但建筑能在玩家战斗、刷题、跑步或切去另一个资源时提供底层积累。

### 6.3 主动效率另开成长线

后续可以新增一个建筑或技能，专门提高主动行为效率，例如：

| 名称 | 类型 | 解锁建议 | 效果 |
|------|------|----------|------|
| 练习计划 | 技能 | 击败 J 题或第 2 图 | 主动收集时额外获得挂机收益的 `10% * level` |
| 计时器 | 建筑 | 第 2 图数学训练后 | 主动收集效率 `+5% * level` |

推荐第一章先不加入主动效率建筑，避免早期系统过多。第二章开始可以用它把“主动操作”和“挂机建筑”连接起来。

### 6.4 乘区边界

第一章建筑不应堆出过大的乘区。

建议边界：

| 类型 | 第一章建议上限 |
|------|----------------|
| 思维力/专注力自动产出 | 单项约 `+0.25/s` 到 `+0.6/s` |
| 基础回血额外值 | 约 `+0.4/s` 到 `+0.8/s` |
| 错题本攻击加成 | 约 `+8%` 到 `+14%`，且只对当前题生效 |
| 记忆书架全局攻血 | 约 `+2%` 到 `+6%` |
| 自动/快进效率 | 约 `+5%` 到 `+15%` |

### 6.5 章节推荐等级

建筑可以无上限，但 UI 和文档应承认“章节推荐等级”：

- 第一章只期望玩家把基础建筑升到 3-7 级。
- 超过推荐等级仍可继续升，但边际收益明显变低。
- 第二章以后可以通过新建筑、新资源或技能树解锁更高效率，而不是逼玩家无限刷第一章建筑。

### 6.6 避免多系统同乘爆炸

第一章建筑尽量少直接给百分比攻血。资源建筑给固定产出，回血建筑给固定恢复，错题本只针对当前未完成题目的局部战斗体验。

全局百分比集中在技能树、成就、稀有卡、记忆书架少量提供。第一章末建筑直接贡献的等效战力提升建议不超过 `1.5x-2x`。

## 七、模拟验证目标

新增 `sim/chapter1_buildings.js` 用来验证：

- 新档到前 5 图通关的预计时间。
- 每图 A/E/J/O/T 五个关键节点的到达时间。
- 各建筑最终等级和效果。
- 战败次数、回血耗时、灵感获取量。
- 刷题模式解锁后，刷题耗时、刷题次数、刷题收益和推关收益占比。
- 是否出现某个建筑收益过强，导致第一章被过快压扁。
- 高等级建筑的成本/收益比是否快速下降，防止无上限爆炸。

建议先比较四种策略：

| 策略 | 说明 |
|------|------|
| `baseline` | 不建新建筑，只使用当前基础循环 |
| `resourceOnly` | 只建草稿桌和安静角落 |

验收目标：

- `baseline` 应能暴露失败等待、重试间隔和中途目标稀薄的问题。
- 第 1 图不应解锁全部建筑，第 2-5 图应持续出现 1-2 个新建筑或建筑相关目标。
- 当前 5 图测试应能在 45-90 分钟区间内完成，且不会在第 2 图后因怪物攻击高过血线上限而硬锁。
- 建筑等级不应在第一章内自然冲到几十级。
- 即使模拟强行投入资源，高等级收益也应因成本指数和效果递减而变差。

注意：模拟器会自动分配资源和升级技能树，跑出的耗时更接近理想策略下限；真实体感需要结合页面操作成本、玩家是否及时升级、是否手动重试等因素继续校准。

当前实现同步调整了第 2 图以后的题目攻击/血量曲线：保留随图递增的指数趋势，并为数学、语文等学科闯关奖励预留数值空间。当前模拟器只覆盖建筑、刷题和基础技能树循环，未完整计入后续学科奖励，所以第 5 图后段出现卡点是可接受的，后续应通过学科奖励、卡片和更高阶建筑填补。
